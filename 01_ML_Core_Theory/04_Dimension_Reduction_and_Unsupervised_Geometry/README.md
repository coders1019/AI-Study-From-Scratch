#### 1. PCA의 투영 분산 극대화: 왜 하필 '분산'이며, SVD와는 어떻게 기하학적으로 커넥션이 일어나는가? (수학적 구조)
* **피타고라스 정리와의 동치성:** 대부분의 교재는 PCA를 "데이터 투영 시 분산이 최대인 축을 찾는 알고리즘"으로만 다루지만, 본 실험에서는 이를 기하학적 정보 손실 최소화 관점에서 증명했습니다. 직각삼각형 구조상 **[원점-원본 샘플 거리²] = [투영 거리²] + [투영 오차²]** 가 성립하며, 원본 거리는 불변하므로 투영 오차(정보 손실)를 최소화하는 행위가 곧 투영 분산을 극대화하는 행위와 완전히 수학적으로 동치($\iff$)됨을 실증했습니다.

#### 2. 점진적 PCA (Incremental PCA): 미니배치의 투영 결과는 어떻게 전체 투영과 일치하는가? (메모리 스케일)
* **온라인 공분산 업데이트:** 메모리에 다 올라가지 않는 대용량 환경에서 `IncrementalPCA`는 랭크-1 업데이트(Rank-1 Update) 방식을 통해 기존 평균과 새로운 평균을 융합하여 전체 공분산 구조를 조금씩 정교화합니다. 가상 고차원 데이터를 활용해 배치 사이즈를 쪼개어 학습시킨 결과, 전체 데이터를 한 번에 본 일반 `PCA` 주성분 벡터와의 코사인 유사도가 **0.999999 이상**으로 수렴하며 메모리 효율성과 정밀도를 동시에 달성함을 확인했습니다.

#### 3. 랜덤 투영(Random Projection)의 마법: 무작위로 튕겨내도 거리가 잘 보존되는 이유 (고차원 기하학)
* **존슨-린덴슈트라우스 정리(JL Lemma):** 고차원 공간에서는 거의 모든 벡터가 서로 직교($90^\circ$)에 가깝다는 기하학적 특성이 존재합니다. 이 덕분에 무작위 축으로 선형 투영을 하더라도 타겟 저차원 $d$가 특정 하한선 이상이면 데이터 간 거리가 원래 거리의 $1 \pm \epsilon$ 범위 내로 안전하게 보존됩니다. 놀랍게도 이 최소 차원 공식에는 원래 차원 $D$가 포함되지 않아, 원본이 100만 차원이든 오직 샘플 수($N$)와 허용 오차($\epsilon$)에 의해서만 하한선이 결정되는 차원의 마법을 실증했습니다.

#### 4 & 5. LLE의 차원 불일치 모순과 목적 함수의 반전: 스프링 시스템과 고유값 선택 원리 (매니폴드 최적화)
* **최소제곱법과 평형 상태:** 고차원에서 독립적이었던 이웃 관계(가중치 $W$)를 좁은 저차원으로 이식할 때 발생하는 기하학적 모순(Overdetermined)을 LLE는 완벽한 복원이 아닌 **'최소제곱법 관점의 전역적 평형 상태'**를 찾는 방식으로 해결합니다. 
* **고유값의 반전:** 분산을 최대화하기 위해 가장 큰 고유벡터를 잡는 PCA와 반대로, LLE는 국소 구조가 파괴되는 '오차(Embedding Cost)'를 최소화해야 하므로 2차 형식 $\text{Tr}(YMY^T)$에서 0을 제외하고 **가장 작은(Smallest) 고유값**에 대응하는 고유벡터를 좌표축으로 선택해야 매니폴드가 매끄럽게 보존됨을 실증했습니다.

#### 실험 1. PCA 투영 분산 극대화 및 피타고라스 정리 기하학적 동치성 검증
* **파일명:** `ch8_1_pca_variance_proof.py`
* **실험 목적 (Why):** 투영 각도를 변화시킬 때 투영 분산의 변화 곡선과 복원 오차 제곱합 곡선이 정확히 역행 관계를 이루는지 확인하고, 두 값의 합이 항상 원점 기준 데이터의 고유 에너지가 됨을 수학적으로 검증합니다.
* **실험 방법 (How):** 평균 중심화(Centering)된 2차원 가상 데이터를 생성한 뒤, $0^\circ$부터 $180^\circ$까지 투영 축 $w$를 회전시키며 투영 데이터의 분산과 복원 오차(Reconstruction Error) 제곱합을 정밀 계측하여 그래프로 시각화합니다.
* **실험 결과 (Result):** 투영 분산이 정점을 찍는 최적의 주성분 축 각도에서 복원 오차가 정확히 최저점을 기록했습니다. 또한 모든 각도에서 [분산 + 오차 제곱합]이 완벽한 직선(상수)을 유지함으로써 피타고라스 정리에 기반한 PCA의 최적화 메커니즘을 눈으로 증명했습니다.

#### 실험 2. 점진적 PCA와 일반 배칭 PCA의 주성분 축 수렴도 정량 평가
* **파일명:** `ch8_2_incremental_pca_validation.py`
* **실험 목적 (Why):** 미니배치 단위로 나누어 공분산 구조를 점진적으로 비틀어가는 `IncrementalPCA`가 전체 데이터를 한 번에 연산한 `PCA`와 수학적으로 동일한 축을 찾아내는지 신뢰도를 측정합니다.
* **실험 방법 (How):** 10,000개의 샘플과 50차원으로 구성된 대형 행렬 데이터를 생성한 후, `PCA`와 미니배치 크기를 1,000개로 제한한 `IncrementalPCA`를 각각 학습시켜 도출된 상위 5개 주성분 행렬(Components) 간의 코사인 유사도 대각 성분을 산출합니다.
* **실험 결과 (Result):** 1번부터 5번 주성분 축까지 모든 축에서 **코사인 유사도가 절대값 기준 1.000000에 수렴**하는 완벽한 정렬 상태를 보였습니다. 이는 대용량 데이터 환경에서 물리적 RAM 부담을 획기적으로 줄이면서도 정보 손실 없이 PCA를 수행할 수 있음을 완벽히 증명합니다.

#### 실험 3. 존슨-린덴슈트라우스 정리에 따른 무작위 투영의 거리 보존율 실증
* **파일명:** `ch8_3_random_projection_jl_lemma.py`
* **실험 목적 (Why):** 무작위 가우시안 행렬을 이용해 초고차원 데이터를 압축했을 때, 원본 공간에서의 샘플 간 거리가 깨지지 않고 허용 오차 한계($1 \pm \epsilon$) 내로 방어되는지 분포로 확인합니다.
* **실험 방법 (How):** 샘플 수 $N=500$, 허용 오차 $\epsilon=0.1(10\%)$ 환경에서 JL 하한선 공식이 산출한 타겟 차원 $d$를 계산합니다. 이후 10,000차원의 데이터를 해당 저차원으로 `GaussianRandomProjection`을 수행한 뒤, 투영 전후의 모든 쌍별 거리(Pairwise Distances)의 비율을 히스토그램으로 매핑합니다.
* **실험 결과 (Result):** 10,000차원에서 수천 차원으로 데이터가 무작위 압축되었음에도 불구하고, 압축 후 거리 대 압축 전 거리 비율이 정확히 $1 \pm 0.1$ 범위인 `[0.9, 1.1]` 안전지대 내에 뾰족한 종 모양으로 집중 밀집되는 기하학적 마법을 실증했습니다.

#### 실험 4 & 5. LLE 비용함수 최소화에 따른 고유값 선택 반전 및 매니폴드 보존 실증
* **파일명:** `ch8_4_5_lle_eigenvalue_manifold.py`
* **실험 목적 (Why):** LLE 알고리즘의 최적화 목적식인 Embedding Cost 최소화 과정에서 왜 작은 고유값을 선택해야 하는지 확인하고, 이를 거슬러 큰 고유값을 선택했을 때 국소 구조 사슬 시스템이 어떻게 파괴되는지 대조 분석합니다.
* **실험 방법 (How):** 3차원 스위스 롤(Swiss Roll) 데이터를 복원한 뒤, 내부 정렬 행렬 $M$의 고유값 분해 메커니즘을 모사합니다. 0을 제외하고 가장 작은 고유값 두 개를 선택해 펼친 저차원 임베딩과, 반대로 고주파 오차가 극대화된 가중치 축에 투영된 결과를 2차원 평면 위에 플롯하여 위상 보존 상태를 대조합니다.
* **실험 결과 (Result):** 가장 작은 고유값들을 선택한 경우 말려 있던 곡면이 사슬이 풀리듯 매끄러운 2차원 평면 카펫 형태로 완벽히 언롤링(Unrolling)된 반면, 큰 고유값 방향 혹은 오차 축으로 강제 투영한 결과는 이웃 관계가 사방으로 찢어발겨 지고 뭉개진 최악의 붕괴 패턴을 거동 실측했습니다.
