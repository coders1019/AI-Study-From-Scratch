### 1. 지니 불순도($G$) vs 엔트로피($H$)의 테일러 근사와 기하학적 민감도 격차 (수학적 구조)
- **연산 오버헤드:** 엔트로피의 로그 항($-\ln p_i$)을 $p_i=1$ 주위에서 테일러 전개(Taylor Expansion)하면 지니 불순도 공식($1 - \sum p_i^2$)이 유도됨을 확인했습니다. 즉, 지니는 무거운 초월함수(로그) 연산을 다항식 공간으로 1차 근사하여 컴퓨터의 부담을 줄인 형태이며, 실제 500만 건 부동소수점 배열 실측 결과 엔트로피가 지니 대비 **약 20~30%의 추가 연산 오버헤드**를 발생시켰습니다.
- **불균형 데이터 민감도:** $p_i \to 0$일 때 엔트로피의 음의 로그는 기울기가 무한대로 발산하는 강한 비선형성을 띱니다. 이 기하학적 차이 때문에 엔트로피는 극단적 불균형 환경(99:1)에서 미량의 희귀 노이즈 클래스에 과도하게 반응(발작)하여 순도 100%를 만들기 위해 하위 노드를 강박적으로 쪼개며 깊은(Deep) 과대적합 트리를 형성하는 성향을 실측했습니다.

### 2. 결정 트리 하이퍼파라미터 규제에 따른 결정 경계 실측 (시각화)
- **규제 메커니즘:** 모델이 과대적합되었을 때 격자 모양의 결정 경계 표면을 어떤 레버로 통제해야 하는지 직관적 기준 수립 목적.
- **구조적 검증:** 의도적인 노이즈를 섞은 `make_moons` 데이터셋 공간 위에 파라미터별 결정 경계 표면(`contourf`)을 시각화하여 고유 성격을 규명했습니다.
  - `max_depth`: 트리의 세로축 성장을 강제 제한하여 데이터 전체의 거시적 흐름을 빠르게 잡는 **탑다운(Top-down) 레버**.
  - `min_samples_split`: 노이즈로 인한 자잘한 국소 분할을 단호하게 차단.
  - `min_samples_leaf`: 불균형 데이터 환경에서 소수 샘플만 모인 '고립된 파편화 리프 노드'의 생성을 막는 **정밀 방어 레버**.

### 3. 지니 불순도의 하향 곡선: 부모와 자식 노드의 수학적 역전 현상 (목적함수)
- **수학적 모순 진단:** 트리가 하향 분할될수록 모든 단일 자식 노드의 불순도는 부모보다 항상 낮아질 것이라는 통념 타파.
- **기하학적 해소:** CART 알고리즘이 최소화하는 비용함수 $J(k, t_k)$는 단일 노드의 불순도가 아니라, 부모 대비 **샘플 수 가중치가 반영된 자식 노드 불순도의 합**임을 추적했습니다. 전체 시스템의 정보 이득(Information Gain)을 극대화하기 위해 한쪽 자식 노드로 노이즈 샘플 소수를 몰아넣어 자식의 불순도가 부모보다 오히려 2배 폭발하는 **'수학적 역전 노드'**가 정상 검출됨을 트리 순회 데이터(`tree_.impurity`)로 실증했습니다.

### 4. 배깅(Bagging) vs 페이스팅(Pasting): 중복 허용이 앙상블 분산을 낮추는 원리 (통계학)
- **통계적 de-correlation:** 앙상블의 최종 분산 공식 $\text{Var}(\bar{X}) = \rho\sigma^2 + \frac{1-\rho}{B}\sigma^2$에서 트리의 개수($B$)를 무한히 늘려도 트리 간 상관계수($\rho$)가 살아있으면 전체 분산의 하한선(Floor)이 무너지지 않습니다.
- **구조적 차이:** 비복원 추출(Pasting)은 원본 데이터의 거시적 통계 특성을 공유하므로 개별 트리 간 $\rho$가 높게 유지됩니다. 반면 복원 추출(Bagging)은 중복 샘플링을 통해 데이터 공간의 밀도와 경계를 강제로 왜곡하여 개별 트리가 독립적인 에러 패턴을 가지도록 통계적 연결고리를 완벽히 끊어냅니다. 100개 베이스 트리 간의 예측치 상관관계를 실측하여 배깅이 $\rho$를 극적으로 낮추고 최종 투표(Averaging) 시 무작위 에러를 상쇄시키는 메커니즘을 확인했습니다.

### 5. 현대 앙상블의 파괴적 혁신: XGBoost vs LightGBM (알고리즘 구조)
- **속도 혁신 벤치마킹:** 대용량 데이터셋 환경에서 사이킷런 기본 GBM의 병목 한계를 극복하는 두 현대적 알고리즘의 구조 분석.
- **분할 구조의 차이:** XGBoost는 손실 함수를 테일러 전개하여 1차 미분(Gradient)과 2차 미분(Hessian)을 반영하고 수평 정렬 분할(Level-wise)을 취하는 반면, LightGBM은 기존 고정관념을 깨고 손실을 가장 많이 줄이는 단 하나의 리프 노드만 수직으로 깊게 쪼개는 **리프 중심 분할(Leaf-wise)**과 그레이디언트 기반 샘플링(**GOSS**)을 채택하여 연산 속도를 혁신적으로 단축(XGBoost 대비 수 배 이상의 속도 향상)함을 실측했습니다.

### 실험 1. Gini vs Entropy 연산 스트레스 및 불균형 데이터 민감도 실측
* **파일명:** `01_tree_purity_and_regulation_analyser.ipynb` (`benchmark_purity_and_imbalance`)
* **실험 목적 (Why):**
  * 다항식 기반의 지니 불순도와 로그 초월함수 기반의 엔트로피가 실제로 마이크로 연산 단에서 유발하는 부동소수점 오버헤드 격차를 측정합니다.
  * 극단적 클래스 불균형(99:1) 환경에서 엔트로피의 강한 비선형 기울기가 노드 과분할 및 과대적합을 유도하는지 노드 총개수로 검증합니다.
* **실험 방법 (How):**
  * 500만 건의 난수 부동소수점 배열을 생성한 뒤, 각 수식을 적용하여 소요 시간을 마이크로초 단위로 계측합니다.
  * `make_classification`으로 생성한 5000개 샘플 기반의 99:1 불균형 데이터셋에 max_depth=10 제한을 두고 지니 트리와 엔트로피 트리를 각각 학습시킨 뒤, 생성된 총 노드 개수(`tree_.node_count`)를 비교합니다.
* **실험 결과 (Result):**
  * **연산 비용 차이 확인:** 초월함수 로그 연산 루프를 도는 엔트로피가 지니 불순도 연산 대비 약 20~30%의 확실한 오버헤드를 발생시켰습니다.
  * **과분할 성향 실증:** 엔트로피 트리가 미량의 소수 샘플 분류에 강박적으로 집착하면서 지니 트리보다 훨씬 더 많은 노드(과분할)를 생성하여 불균형 환경에서 오버피팅 위험도가 치명적으로 높아짐을 증명했습니다.

### 실험 2. 하이퍼파라미터 규제 제어에 따른 결정 경계(Contour) 시각화
* **파일명:** `01_tree_purity_and_regulation_analyser.ipynb` (`visualize_regulation_boundaries`)
* **실험 목적 (Why):**
  * `max_depth`와 `min_samples_leaf` 하이퍼파라미터가 모델 복잡도를 억제하고 격자 모양의 오버피팅 경계를 완화하는 물리적 양상을 눈으로 확인합니다.
  * 복잡하게 얽힌 격자 경계가 규제 레버를 돌림에 따라 어떻게 부드럽고 굳건한 평면으로 변모하는지 대조 분석합니다.
* **실험 방법 (How):**
  * `make_moons` 데이터셋(샘플 300개, 노이즈 0.3)을 활용하여 규제가 없는 모델(Unregulated), `max_depth=4` 모델, `min_samples_leaf=10` 모델을 각각 피팅합니다.
  * 2차원 평면 위에 결정 경계 표면(`contourf`)을 1행 3열 플롯으로 매핑하여 결정 경계가 완화되는 양상을 대조 시각화합니다.
* **실험 결과 (Result):**
  * 규제가 없는 모델은 노이즈 하나하나에 반응하여 지저분한 격자 노이즈(Grid Noise)를 보였으나, `max_depth`에 의해 전역적 흐름 위주로 거시 제어되거나, `min_samples_leaf`에 의해 소수 샘플에 피팅된 고립 영역(Local Fragmentation)이 단호하게 차단되는 거동을 실측 시각화했습니다.

### 실험 3. CART 가중합산 목적함수에 따른 자식 노드 불순도 역전 추적
* **파일명:** `01_tree_purity_and_regulation_analyser.ipynb` (`track_purity_inversion`)
* **실험 목적 (Why):**
  * "트리는 아래로 분할될수록 모든 단일 자식 노드의 불순도가 낮아진다"는 직관적 오해를 수식과 실측 데이터로 타파합니다.
  * CART 알고리즘이 시스템 전체의 정보 이득(Information Gain) 극대화를 위해 국소적인 불순도 상승을 의도적으로 허용하는지 검증합니다.
* **실험 방법 (How):**
  * 90:10 비율의 불균형 데이터(1,000개 샘플, 특성 5개)를 주입한 결정 트리(`max_depth=5`)를 빌드합니다.
  * 내부 분할 노드를 전수 순회(`tree_.children_left`, `tree_.impurity`, `tree_.n_node_samples`)하면서 부모 노드의 지니 불순도보다 자식 노드의 지니 불순도가 오히려 상승한 '역전 노드'를 탐색하고 샘플 가중치를 역추적합니다.
* **실험 결과 (Result):**
  * 부모 노드의 지니 불순도(ex. 0.24)보다 왼쪽 자식 노드의 불순도(ex. 0.48)가 오히려 2배 가량 폭발하는 **'수학적 역전 노드'**들이 선명하게 검출되었습니다.
  * 이는 샘플 수가 극히 적은 자식 쪽으로 노이즈를 전부 몰아내는 대신, 반대쪽 대다수 샘플 노드를 완벽히 정화($G \to 0$)하여 가중합 손실을 최소화하려는 알고리즘의 최적화 희생 매커니즘임을 실증했습니다.

### 실험 4. 복원(Bagging) vs 비복원(Pasting) 추출에 따른 트리 간 상관계수($\rho$) 격차 실측
* **파일명:** `02_ensemble_variance_and_boosting_bench.ipynb` (`evaluate_ensemble_variance_floor`)
* **실험 목적 (Why):**
  * 단순히 중복 샘플링을 허용하느냐 마느냐의 한 줄 차이가 전체 앙상블의 분산($\text{Var}$)을 극적으로 통제하는 통계적 원리를 규명합니다.
  * 개별 트리들이 독립적인 에러 패턴을 가지도록 끊어내는 '상관관계 해제(De-correlation)' 현상을 수치로 입증합니다.
* **실험 방법 (How):**
  * 1,500개 샘플과 20개 특성을 가진 데이터셋 환경에서 50개의 트리를 가진 `BaggingClassifier`를 각각 복원 모드(`bootstrap=True`)와 비복원 모드(`bootstrap=False`)로 학습시킵니다.
  * 각 베이스 트리들의 개별 예측치 행렬을 도출한 후, 상관계수 행렬(`np.corrcoef`)의 평균값($\rho$)을 산출하여 대조합니다.
* **실험 결과 (Result):**
  * 페이스팅 트리는 원본의 거시적 통계 특성을 그대로 공유하여 트리 간 평균 상관계수 $\rho$가 매우 높게 유지된 반면, 배깅 트리는 중복 허용 샘플링에 의한 데이터 공간 밀도 왜곡 덕분에 상관계수 $\rho$가 극적으로 낮게 측정되었습니다. 통계학적 공식의 좌항($\rho\sigma^2$)인 분산 하한선을 파괴하는 원리를 실증 수치로 증명했습니다.

### 실험 5. 현대 하이테크 부스팅 엔진 실측 및 분할 알고리즘 벤치마킹
* **파일명:** `02_ensemble_variance_and_boosting_bench.ipynb` (`benchmark_modern_boosting_engines`)
* **실험 목적 (Why):**
  * 대용량 데이터 트랙션 환경에서 사이킷런 기본 GBM이 유발하는 극심한 연산 병목을 극복하는 현대 앙상블 엔진의 파괴적 혁신을 실측합니다.
  * Level-wise(수평 균형 중심) 구조와 Leaf-wise(수직 손실 중심) 구조가 유발하는 하드웨어 연산 효율성의 격차를 증명합니다.
* **실험 방법 (How):**
  * 연산 스트레스를 위해 40,000개의 샘플과 50개의 특성을 가진 헤비 가상 회귀 데이터셋을 생성합니다.
  * 동일한 하이퍼파라미터 조건(`n_estimators=100, max_depth=6, n_jobs=-1`) 하에 `XGBRegressor`와 `LGBMRegressor`를 전수 훈련시키고 학습 완료 성능을 초 단위로 정밀 계측합니다.
* **실험 결과 (Result):**
  * 테일러 전개 기반의 1·2차 미분(Gradient & Hessian) 연산 및 수평 확장을 취하는 XGBoost 대비, 손실 함수 감소율이 가장 극대화되는 단 하나의 리프 노드만 수직으로 빠르게 깊게 파고드는 LightGBM이 압도적인 속도(0.1초 대 수렴)를 기록했습니다. GOSS와 Leaf-wise 메커니즘이 결합되어 수 배 이상의 탁월한 연산 효율성 혁신을 이뤄냄을 데이터로 확인했습니다.

### 2) 앙상블 무작위성 및 현대 부스팅 엔진 연산 효율성 ($m = 40,000, n = 50$)

| :--- | :--- | :--- | :--- |
| **Pasting Ensemble (`bootstrap=False`)** | 비복원 추출 기반 샘플링 | 높은 트리 간 상관계수 ($\rho$) 기록 | 거시 특성 공유로 독립 에러 패턴 형성이 제한적임 |
| **Bagging Ensemble (`bootstrap=True`)** | 복원 추출 (중복 허용 왜곡) | **극적으로 낮은 상관계수 ($\rho$)** | 트리 간 동질성을 파괴(De-correlation)하여 분산 하한선 하락 |
| **XGBoost Regressor** | Level-wise (수평 분할) & Hessian | 수 초 대 연산 지연 발생 | 2차 미분 기반 정밀도는 높으나 대용량 트랙션에서 정렬 병목 |
| **LightGBM Regressor** | Leaf-wise (수직 분할) & GOSS | **0.1초 대 초고속 수렴 완료** | 비대칭 최적 리프 분할 구조를 통한 파괴적 자원 최적화 |
