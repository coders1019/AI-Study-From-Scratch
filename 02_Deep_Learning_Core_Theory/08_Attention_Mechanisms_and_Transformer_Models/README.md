##  핵심 트러블슈팅 요약

### 1. 동일한 행렬의 분화: 어떻게 하나의 입력이 서로 다른 Q, K, V로 작동할까?
* **유발된 의문:** 셀프 어텐션(Self-Attention) 레이어에 입력되는 토큰 행렬 $X$와 초기 가중치 행렬들이, 어떻게 학습을 거치면서 각각 '질문(Q)', '키워드(K)', '의미 정보(V)'라는 고유하고 이질적인 기능으로 분화될 수 있는가?
* **수학적·구조적 본질:** 가중치 행렬이 입력 공간을 서로 다른 서브스페이스(Subspace)로 사영하며, 어텐션 공식 내부의 연산 위치에 따라 세 행렬에 흐르는 그래디언트의 형태가 완전히 다르게 설정됩니다. $W_Q, W_K$는 상호작용 공간($Q K^T$)에서 상대적 관계성을 학습하고, $W_V$는 소프트맥스 지도에 따라 의미적 콘텐츠를 가공하는 독립된 차원으로 분화(**Symmetry Breaking**)됩니다.

### 2. 멀티헤드 어텐션의 차원 쪼개기: 단어의 일부분만으로 유의미한 학습이 가능할까?
* **유발된 의문:** 전체 임베딩 차원($d_{model}=512$)을 헤드의 개수($h=8$)로 나누어 고작 64차원의 벡터들로 쪼개면, 단어가 가진 본래의 풍부한 세만틱(Semantic) 정보가 훼손되거나 손실되는 것 아닌가?
* **수학적·구조적 본질:** 차원을 쪼개는 것은 정보의 손실이 아니라 서로 다른 기하학적 관점에서 데이터를 바라보게 하는 공간의 분할입니다. 64차원은 $2^{64}$의 표현력을 지녀 단일 규칙(Aspect) 하나를 추적하기에 차고 넘치며, 이 독립적인 해석 결과물들을 마지막에 `Concat`하여 거대한 숲을 스캔하는 독자적인 **앙상블(Ensemble) 체계**를 완성합니다.

### 3. 포지셔널 인코딩의 벡터 덧셈: 왜곡의 위험을 우회하는 수학적 마법
* **유발된 의문:** 단어 고유의 벡터 공간 위에 위치 벡터 값을 그냥 성분별로 무작위로 더해버리면($X = E + P$), 기존 단어의 세만틱 정보가 오염되거나 왜곡될 위험이 크지 않은가? 왜 콘캣(`Concat`)이 아닌 `Add`인가?
* **수학적·구조적 본질:** **고차원 공간의 직교성(Orthogonality)**과 선형 사영 변환 덕분입니다. 고차원 공간에서 무작위로 뽑은 두 벡터는 직교($90^\circ$)할 확률이 극도로 높으므로 신호가 치명적으로 침범하지 않으며, 입력 변환 과정에서 가중치 행렬 $W$가 개입하여 두 신호($E\cdot W + P\cdot W$)를 수식적으로 완벽히 분리해 냅니다. 이를 통해 `Concat` 대비 연산량을 혁신적으로 아끼게 됩니다.

### 4. 크로스 어텐션의 주객전도: 왜 '질문(Q)'의 주체는 디코더여야만 할까?
* **유발된 의문:** 정보량이 더 많고 정교한 인코더가 주체가 되어 디코더에게 정보를 주입($Q$=인코더)해야 하는 것 아닌가? 왜 굳이 이제 막 첫 단어를 뱉으려는 디코더가 $Q$를 던지고 인코더는 수동적으로 반응하는가?
* **수학적·구조적 본질:** **'듣는 이(디코더)'의 맥락에 맞춘 정보의 필터링(Information Filtering)**입니다. 디코더가 $Q$를 던져야만 현재 디코더가 처한 생성 상황에 정확히 일치하는 정보만 인코더($K$)에게 요구하고, 필요한 핵심 의미($V$)만 압축해서 가져와 맥락 오염을 방지할 수 있습니다.

##  핵심 실증 실험 (Experiments & Results)

###  실험 1. 가중치 분화: 기하학적 궤적 변위 및 쌍성계(Binary System) 분석
* **파일명:** `01_gradient_anisotropy_and_qkv_differentiation.ipynb`
* **실험 목적 (Why):** 문맥적 상호작용 연산 하에서 역전파를 수행할 때, $W_Q, W_K, W_V$ 가중치 행렬이 실제로 각자 독립된 기하학적 경로로 학습 및 분화되는지 궤적 사잇각을 통해 실측합니다.
* **실험 방법 (How):** 단층 셀프 어텐션 연산 구조를 재현하고 가상 역전파를 진행합니다. 초기 가중치 상태 대비 학습 후 이동한 방향 벡터들($\Delta W_Q, \Delta W_K, \Delta W_V$) 간의 코사인 유사도(Cos-Sim)를 측정하여 기하학적 분화 패턴을 분석합니다.
* **실험 결과 (Result):** 학습 초기 대칭 상태가 파괴(Symmetry Breaking)된 이후, $Q$와 $K$는 상호 관계 맵을 형성하기 위해 특정 역학적 평면 안에서 움직이는 반면, $V$는 관계성 연산에 간섭받지 않고 완전히 독립된 다른 차원의 공간으로 분화됨을 증명했습니다.

### 실험 2. 멀티헤드 차원 분할: 헤드별 독립 부분공간 텍스트 맵 및 유효 랭크 검증
* **파일명:** `02_multihead_subspace_tiling_and_effective_rank.ipynb`
* **실험 목적 (Why):** 전체 임베딩 공간을 8개 헤드의 저차원($d_k=64$)으로 쪼갰을 때, 각 헤드가 서로 간섭 없이 독립된 서브스페이스를 점유하는지, 그리고 이들이 모여 정보 손실 없는 최종 어텐션 맵을 형성하는지 물리적으로 검증합니다.
* **실험 방법 (How):** 헤드별 고유한 부분공간 편향을 부여한 후 어텐션 맵을 생성합니다. 헤드 간 어텐션 확률 분포의 코사인 유사도를 그리드 격자로 시각화하고, 전체 헤드를 평균하여 결합된 최종 숲(Combined Attention Map)의 유효 행렬 랭크(Matrix Rank) 차원 수를 계산합니다.
* **실험 결과 (Result):** 헤드 간 상호 중첩도 분석 결과 대부분의 헤드가 독립적(`..` 표시) 영역을 유지함을 시각적으로 확인했으며, 결합된 맵의 유효 랭크 차원 수(`Effective Rank = 12`)를 실측하여 숲을 완벽히 이해하는 독자적인 앙상블 체계를 증명했습니다.

### 실험 3. 포지셔널 덧셈: 고차원 무작위 직교성(Orthogonality)의 임계점 추적
* **파일명:** `03_high_dim_orthogonality_and_additive_separability.ipynb`
* **실험 목적 (Why):** 저차원부터 초고차원 LLM 영역까지 단어 임베딩 공간 $E$와 위치 공간 $P$의 벡터를 단순히 더했을 때 발생하는 신호 간섭도(Cross-Talk)와 실제 사잇각의 변화를 추적하여 덧셈 설계의 타당성을 선형대수학적으로 입증합니다.
* **실험 방법 (How):** 차원 크기($d_{model}=16$에서 $2048$)별로 무작위 벡터 샘플 500개를 생성한 후, 두 신호 간의 평균 사잇각(Degree)과 덧셈 공간($X=E+P$)에서 원래 임베딩 $E$가 가지는 방향성 훼손율(Interference %)을 정밀 측정합니다.
* **실험 결과 (Result):** 차원이 512를 넘어서는 순간 고차원 기하학 법칙에 의해 두 벡터의 평균 사잇각은 **`90.0°` (완전직교)**에 대치되며 신호 간섭도가 `29.3%` 수준으로 완벽히 안정화됨을 확인했습니다. 즉, 가중치 행렬 $W$가 이들을 독립된 두 축(Basis)으로 분리해낼 수 있는 최적의 임계점을 수치적으로 실측했습니다.

### 실험 4. 크로스 어텐션 주체성: 주객전도 시 발생하는 정보 소실율 및 효율성 검증
* **파일명:** `04_cross_attention_decoder_query_information_gain.ipynb`
* **실험 목적 (Why):** 질의(Query)의 주체를 인코더와 디코더로 각각 설정했을 때 발생하는 정보이론적 효율성을 대조하여, 디코더가 $Q$여야만 하는 구조적 당위성을 물리 메모리와 에너지 분산 관점에서 실증합니다.
* **방법 (How):** 긴 인코더 텐서와 짧은 디코더 텐서를 활용해 정석 구조($Q$=디코더)와 주객전도 구조($Q$=인코더)를 구축합니다. 균등 분포 대비 어텐션 확률이 얼마나 예리하게 타겟팅되는지 정보 이득(Information Gain) 스펙트럼과 엔트로피를 측정합니다.
* **실험 결과 (Result):** 정석 구조는 맥락 압축 이득 **`1.1074` (선택적 필터링 작동)**를 기록한 반면, 주객전도 구조는 `0.6502`로 급감했습니다. 인코더가 질문을 던질 경우 빈약한 디코더 공간에 모든 인코더 토큰이 밀고 들어와 **노이즈 범람 및 맥락 오염(Dissipation)**을 유발함을 실측 증명했습니다.

## 핵심 소스코드

### PyTorch LSTM 내부 차원 기하학 물리적 증명 모듈 (`exp02_pytorch_lstm_dim_geometry.py`)

```python
import torch
import torch.nn as nn

# 1. 하이퍼파라미터 정의 (다층 + 양방향 모델 구조 강제)
seq_len = 5
batch_size = 2
input_size = 4
hidden_size = 3
num_layers = 2

# Bidirectional 모델 선언
bi_lstm = nn.LSTM(
    input_size=input_size, 
    hidden_size=hidden_size, 
    num_layers=num_layers, 
    bidirectional=True
)

# 더미 시퀀스 데이터 생성 (seq_len, batch_size, input_size)
X = torch.randn(seq_len, batch_size, input_size)

# 순전파 실행
output, (h_n, c_n) = bi_lstm(X)

print("=== [실험 2] 텐서 기하학적 슬라이싱 매치 ===")
print(f"Output Shape : {list(output.shape)}")  # [5, 2, 6] -> 6 = 2(방향) * 3(hidden)
print(f"H_n Shape    : {list(h_n.shape)}")     # [4, 2, 3] -> 4 = 2(레이어) * 2(방향)

print("\n--- 수학적/물리적 일치성 검증 ---")

# 가설 1: output의 최후 타임스텝(T=-1) 내 정방향(Forward) 벡터는 
#        h_n의 최상위 레이어(Layer 2)의 정방향 인덱스 메모리와 완벽히 일치해야 한다.
top_layer_forward_hn = h_n[2, :, :] 
last_timestep_output_forward = output[-1, :, :hidden_size]
is_matching = torch.allclose(top_layer_forward_hn, last_timestep_output_forward, atol=1e-6)
print(f"▶ 최상위 레이어 [정방향] 일치 여부: {is_matching}")

# 가설 2: 역방향(Backward)의 최종 상태는 시퀀스의 시작점(T=0)에 기록된다.
top_layer_backward_hn = h_n[3, :, :]
first_timestep_output_backward = output[0, :, hidden_size:]
is_backward_matching = torch.allclose(top_layer_backward_hn, first_timestep_output_backward, atol=1e-6)
print(f"▶ 최상위 레이어 [역방향] 일치 여부: {is_backward_matching}")
